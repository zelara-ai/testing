# Zelara Testing Module

Testing and diagnostic module for Zelara.

## Overview

Provides diagnostic tools for testing device-to-device communication, image processing, and cross-platform features.

**Features**:
- Image inversion test (Mobile → Desktop → Mobile)
- WebSocket connection diagnostics
- Device linking validation
- Performance benchmarking

**Unlock**: 0 points (available immediately for development/testing)

## Tech Stack

- TypeScript (React components for mobile/desktop)
- WebSocket (device communication)
- @zelara packages (shared types, state)

For detailed technical documentation, see [wiki](wikis/).
